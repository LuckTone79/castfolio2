## 서비스 개요

<AccordionGroup>
  <Accordion title="방송인이 직접 사용하는 서비스인가요?">
    아닙니다. Castfolio는 방송인 PR 페이지를 제작·납품하는 **에이전트(제작 파트너)**를 위한 B2B SaaS입니다. 방송인 본인이 직접 가입하거나 사용하는 서비스가 아닙니다.
  </Accordion>
  <Accordion title="어떤 포지션의 방송인을 지원하나요?">
    아나운서, 쇼호스트, MC, 리포터 4가지 포지션을 지원합니다. 각 포지션에 맞는 섹션 구성이 제공됩니다.
  </Accordion>
  <Accordion title="테마는 몇 가지인가요?">
    7가지 전문 테마를 제공합니다: 앵커 클린, 프레스티지 블랙, 럭스 마블, 프레시 스카이, 소프트 블러시, 내추럴 그린, 웜 코랄.
  </Accordion>
</AccordionGroup>

## 요금 & 정산

<AccordionGroup>
  <Accordion title="월정액이 있나요?">
    없습니다. Castfolio는 별도 월정액 없이 성과 기반 수수료(15%)만 부과합니다.
  </Accordion>
  <Accordion title="결제는 어떻게 이루어지나요?">
    현재 MVP에서는 오프라인 결제를 지원합니다. 담당자가 직접 결제를 안내하고 확인합니다.
  </Accordion>
  <Accordion title="수수료는 얼마인가요?">
    기본 수수료는 15%입니다. 우수 파트너에게는 할인 혜택이 제공됩니다.
  </Accordion>
  <Accordion title="정산은 언제 이루어지나요?">
    매월 1일에 전월 수익의 85%가 등록된 계좌로 이체됩니다. 최소 정산 금액은 ₩10,000이며, 미달 시 다음 달로 이월됩니다.
  </Accordion>
</AccordionGroup>

## 기능

<AccordionGroup>
  <Accordion title="다국어 지원이 되나요?">
    한국어·영어·중국어 3개 언어를 지원합니다. 공개 PR 페이지에서 방문자가 직접 언어를 전환할 수 있습니다.
  </Accordion>
  <Accordion title="QR 코드는 어떤 형식으로 제공되나요?">
    PNG(300dpi)와 SVG 형식으로 다운로드할 수 있습니다. 명함 인쇄에 바로 사용할 수 있는 고해상도 파일입니다.
  </Accordion>
  <Accordion title="배포 후 수정이 가능한가요?">
    배포 후에도 빌더에서 내용을 수정하고 재배포할 수 있습니다. URL과 QR 코드는 변경되지 않습니다.
  </Accordion>
  <Accordion title="방송인이 검토 링크를 통해 직접 수정을 요청할 수 있나요?">
    검토 링크는 조회 전용입니다. 방송인은 링크로 미리보기만 확인할 수 있으며, 수정 요청은 담당자에게 직접 전달해야 합니다. 담당자가 빌더에서 반영 후 재발송합니다.
  </Accordion>
</AccordionGroup>
