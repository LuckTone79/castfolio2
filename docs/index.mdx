## Castfolio란?

**Castfolio**는 방송인 PR 페이지를 제작·납품하는 에이전트(제작 파트너)를 위한 **B2B SaaS**입니다.

방송인이 직접 사용하는 서비스가 아닙니다. 아나운서, 쇼호스트, MC, 리포터의 PR 페이지를 만들어 납품하는 **전문 제작 파트너**를 위한 플랫폼입니다.

## 사용 흐름

<Steps>
  <Step title="방송인 등록">
    이름, 포지션(아나운서/쇼호스트/MC/리포터), 연락처를 입력합니다.
  </Step>
  <Step title="자료 수집">
    방송인에게 제출 링크를 발송하거나 담당자가 직접 자료를 입력합니다.
  </Step>
  <Step title="PR 페이지 제작">
    7가지 테마 중 하나를 선택하고 빌더로 편집합니다. 평균 10분 이내 완성.
  </Step>
  <Step title="검토 & 결제">
    비공개 미리보기 링크를 방송인에게 전달하고, 확인 후 견적서를 발송합니다.
  </Step>
  <Step title="배포 & 납품">
    공개 링크 + 고해상도 QR 코드 + PDF 명함을 자동 생성하여 납품합니다.
  </Step>
  <Step title="월 정산">
    매월 1일, 수익의 85%가 자동으로 정산됩니다.
  </Step>
</Steps>
