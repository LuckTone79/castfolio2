## 빌더 구성

PR 페이지 빌더는 섹션 단위로 페이지를 구성합니다. 각 섹션을 자유롭게 추가·삭제·순서 변경할 수 있습니다.

## 기본 섹션

| 섹션 | 내용 |
|------|------|
| 히어로 | 이름, 포지션, 대표 사진, 한 줄 소개 |
| 소개 | 상세 프로필 및 자기소개 |
| 방송 경력 | 출연 프로그램 목록 (기간, 채널, 역할) |
| 수상 이력 | 수상 내역 목록 |
| 미디어 | 방송 클립, 사진 갤러리 |
| 연락처 | 이메일, 소속사, SNS 링크 |

## 커스텀 섹션

<Tip>
  기본 섹션 외에도 자유 텍스트, 이미지, 링크 섹션을 추가할 수 있습니다.
</Tip>

## 편집 흐름

<Steps>
  <Step title="테마 선택">
    7가지 테마 중 방송인 이미지에 맞는 테마를 선택합니다.
  </Step>
  <Step title="섹션 구성">
    기본 섹션을 활성화하고, 필요한 커스텀 섹션을 추가합니다.
  </Step>
  <Step title="내용 입력">
    각 섹션에 수집된 자료를 입력하고 배치합니다.
  </Step>
  <Step title="미리보기 확인">
    실시간 미리보기로 최종 결과물을 확인합니다.
  </Step>
</Steps>

## 출력물

빌더에서 배포하면 다음이 자동 생성됩니다.

- **공개 PR 페이지** — `castfolio.app/p/{고유ID}` 형식의 퍼블릭 URL
- **QR 코드** — PNG(300dpi) / SVG 형식 다운로드
- **PDF 명함** — A4 및 명함 사이즈 (86×54mm)
