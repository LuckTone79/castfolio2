## 7가지 전문 테마

각 테마는 방송인의 포지션과 이미지를 고려하여 설계되었습니다.

<AccordionGroup>
  <Accordion title="앵커 클린" icon="circle" defaultOpen={true}>
    흰 배경과 깔끔한 타이포그래피. 뉴스 앵커, 보도 리포터에게 적합합니다.
    신뢰감과 전문성을 강조하는 스타일입니다.
  </Accordion>
  <Accordion title="프레스티지 블랙" icon="circle">
    다크 배경의 고급스러운 레이아웃. 경력 있는 아나운서, 프리미엄 쇼호스트에게 어울립니다.
    무게감과 권위를 표현합니다.
  </Accordion>
  <Accordion title="럭스 마블" icon="circle">
    대리석 텍스처와 골드 포인트. 홈쇼핑 쇼호스트, 럭셔리 브랜드 MC에게 적합합니다.
    고품격 이미지를 연출합니다.
  </Accordion>
  <Accordion title="프레시 스카이" icon="circle">
    밝은 하늘색 계열. 생방송 MC, 엔터테인먼트 리포터에게 어울립니다.
    에너지와 친근함을 표현합니다.
  </Accordion>
  <Accordion title="소프트 블러시" icon="circle">
    따뜻한 핑크-베이지 계열. 뷰티·라이프스타일 쇼호스트, 여성 MC에게 적합합니다.
    감성적이고 세련된 이미지를 연출합니다.
  </Accordion>
  <Accordion title="내추럴 그린" icon="circle">
    자연스러운 그린 계열. 환경·건강 관련 방송 리포터에게 어울립니다.
    신선하고 건강한 이미지를 표현합니다.
  </Accordion>
  <Accordion title="웜 코랄" icon="circle">
    따뜻한 코랄-오렌지 계열. 활기찬 홈쇼핑 쇼호스트, 젊은 MC에게 적합합니다.
    생동감과 친밀감을 전달합니다.
  </Accordion>
</AccordionGroup>

<Note>
  테마는 배포 전까지 언제든지 변경할 수 있습니다. 같은 자료로 여러 테마를 미리보기하여 비교할 수 있습니다.
</Note>
