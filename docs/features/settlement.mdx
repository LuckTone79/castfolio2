## 수수료 구조

Castfolio는 월정액 없이 **성과 기반 수수료**만 부과합니다.

| 항목 | 내용 |
|------|------|
| 플랫폼 수수료 | **15%** |
| 파트너 수익 | **85%** |
| 정산일 | 매월 1일 |
| 최소 정산 금액 | ₩10,000 |
| 정산 방법 | 계좌 이체 |

<Info>
  최소 정산 금액(₩10,000) 미달 시 다음 달로 이월됩니다.
</Info>

## 정산 흐름

<Steps>
  <Step title="건별 수익 발생">
    방송인 PR 페이지 납품 완료 및 결제 확인 시 수익이 기록됩니다.
  </Step>
  <Step title="월말 정산 집계">
    매월 말일 기준으로 해당 월의 누적 수익이 집계됩니다.
  </Step>
  <Step title="익월 1일 지급">
    다음 달 1일에 등록된 계좌로 자동 이체됩니다.
  </Step>
</Steps>

## 우수 파트너 혜택

<Tip>
  일정 실적 이상의 우수 파트너에게는 수수료 할인 혜택이 제공됩니다. 자세한 내용은 담당자에게 문의하세요.
</Tip>

## 수익 현황

대시보드의 **정산 현황** 탭에서 다음을 확인할 수 있습니다.

- 이번 달 누적 수익
- 전체 납품 건수
- 월별 수익 추이
- 정산 내역 다운로드 (CSV)
