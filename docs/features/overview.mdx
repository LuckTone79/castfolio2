## 핵심 기능

Castfolio는 방송인 PR 페이지 제작의 전체 워크플로우를 하나의 플랫폼에서 처리합니다.

<CardGroup cols={2}>
  <Card title="자료 수집" icon="inbox">
    방송인이 직접 제출하거나 담당자가 대행 입력. 사진, 방송 경력, 수상 이력, 연락처를 구조화된 폼으로 수집.
  </Card>
  <Card title="PR 페이지 빌더" icon="paintbrush">
    7가지 전문 테마 + 드래그 앤 드롭 빌더. 섹션 추가·삭제·순서 변경 자유롭게.
  </Card>
  <Card title="미리보기 검토" icon="eye">
    토큰 기반 비공개 링크로 방송인과 소통. 공개 전 검토 및 수정 반영.
  </Card>
  <Card title="견적 & 결제" icon="receipt">
    자동 견적서 생성 및 발송. 담당자가 결제 확인 후 배포 진행.
  </Card>
  <Card title="납품 패키지" icon="package">
    공개 PR 페이지 URL + 고해상도 QR 코드(PNG/SVG) + PDF 명함 자동 생성.
  </Card>
  <Card title="월 정산" icon="wallet">
    매월 1일 자동 정산. 수익의 85%를 파트너에게 지급. 최소 정산 ₩10,000.
  </Card>
</CardGroup>

## 다국어 지원

공개 PR 페이지는 3개 언어를 지원합니다.

| 언어 | 지원 여부 |
|------|-----------|
| 한국어 | 지원 |
| 영어 | 지원 |
| 중국어 | 지원 |

방문자가 페이지 상단에서 언어를 직접 전환할 수 있습니다.
