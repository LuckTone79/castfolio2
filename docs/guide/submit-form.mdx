## 제출 링크 생성

방송인 상세 페이지에서 **제출 링크 생성** 버튼을 클릭합니다.

링크는 해당 방송인에게만 유효하며, 만료 기간을 설정할 수 있습니다.

## 방송인 입력 항목

방송인이 제출 링크에서 직접 입력하는 내용입니다.

| 항목 | 필수 여부 | 설명 |
|------|-----------|------|
| 프로필 사진 | 필수 | 대표 사진 1장 (최대 5MB) |
| 한 줄 소개 | 필수 | 50자 이내 |
| 방송 경력 | 필수 | 프로그램명, 채널, 기간, 역할 |
| 수상 이력 | 선택 | 수상명, 주관 기관, 연도 |
| SNS | 선택 | 인스타그램, 유튜브 등 |
| 연락처 | 선택 | 이메일, 소속사 |

## 제출 완료 후

방송인이 제출을 완료하면:

1. 담당자에게 이메일 알림이 발송됩니다
2. 대시보드에서 **자료 수집 완료** 상태로 변경됩니다
3. 빌더에서 바로 PR 페이지 제작을 시작할 수 있습니다

<Warning>
  제출 링크는 방송인과 1:1로 연결됩니다. 링크를 다른 사람에게 공유하지 않도록 안내하세요.
</Warning>
