## 검토 링크란?

검토 링크는 배포 전 방송인이 PR 페이지를 비공개로 확인할 수 있는 임시 링크입니다.

- 검색 엔진에 노출되지 않습니다
- 링크를 아는 사람만 접근할 수 있습니다
- 만료 기간을 설정할 수 있습니다

## 검토 링크 생성

빌더 상단의 **검토 링크 생성** 버튼을 클릭합니다.

생성된 링크 형식: `castfolio.app/preview/{토큰}`

## 발송 방법

<Tabs>
  <Tab title="카카오톡">
    링크를 복사하여 카카오톡으로 방송인에게 직접 전달합니다.
  </Tab>
  <Tab title="이메일">
    대시보드에서 **이메일로 발송** 버튼을 클릭하면 자동으로 안내 이메일이 발송됩니다.
  </Tab>
</Tabs>

## 수정 반영 흐름

<Steps>
  <Step title="링크 발송">
    검토 링크를 방송인에게 전달합니다.
  </Step>
  <Step title="피드백 수신">
    방송인에게 수정 요청을 받습니다 (카카오톡, 이메일 등).
  </Step>
  <Step title="빌더에서 수정">
    빌더에서 요청 내용을 반영합니다. 저장하면 검토 링크에도 즉시 반영됩니다.
  </Step>
  <Step title="재확인 요청">
    수정 완료 후 방송인에게 다시 확인을 요청합니다.
  </Step>
</Steps>

<Note>
  검토 링크는 재생성하지 않아도 됩니다. 빌더에서 수정하면 같은 링크에서 최신 내용이 표시됩니다.
</Note>

## 검토 승인 후

방송인의 최종 승인을 받은 후 견적서를 발송하고, 결제 확인 후 배포합니다.
