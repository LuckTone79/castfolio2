## 전체 흐름

```
방송인 등록 → 자료 수집 → PR 페이지 제작 → 검토 & 결제 → 배포 & 납품 → 월 정산
```

## 단계별 상세 안내

### 1. 방송인 등록

대시보드에서 **방송인 추가**를 클릭합니다.

필수 입력 항목:
- 이름
- 포지션 (아나운서 / 쇼호스트 / MC / 리포터)
- 연락처

### 2. 자료 수집

<Tabs>
  <Tab title="방송인 직접 제출">
    **제출 링크 생성** 버튼을 클릭하여 개인화된 링크를 만들고, 방송인에게 발송합니다.

    방송인이 입력하는 항목:
    - 프로필 사진 (대표 1장 + 추가 최대 5장)
    - 한 줄 소개
    - 방송 경력 목록
    - 수상 이력
    - SNS 및 연락처
  </Tab>
  <Tab title="담당자 직접 입력">
    방송인에게 자료를 받아 담당자가 직접 입력합니다.

    빠른 입력을 위해 엑셀 업로드 기능을 제공합니다 (방송 경력 일괄 입력).
  </Tab>
</Tabs>

### 3. PR 페이지 제작

1. 테마를 선택합니다 (7가지 중 선택)
2. 섹션을 구성합니다
3. 내용을 입력합니다
4. 실시간 미리보기로 확인합니다

평균 제작 시간: **10분 이내**

### 4. 검토 & 결제

<Steps>
  <Step title="검토 링크 발송">
    비공개 검토 링크를 생성하여 방송인에게 카카오톡/이메일로 발송합니다.
  </Step>
  <Step title="피드백 수정">
    방송인의 수정 요청을 빌더에서 반영합니다.
  </Step>
  <Step title="견적서 발송">
    확정 후 자동 생성된 견적서를 발송합니다.
  </Step>
  <Step title="결제 확인">
    입금 확인 후 대시보드에서 결제 완료를 처리합니다.
  </Step>
</Steps>

### 5. 배포 & 납품

**배포** 버튼을 누르면 즉시 생성됩니다:

- 공개 PR 페이지 URL
- QR 코드 (PNG/SVG)
- PDF 명함

납품 방식은 파트너가 직접 결정합니다. 링크 전달, 파일 전달, 또는 인쇄 명함 배송 등.

### 6. 월 정산

별도 작업 없이 매월 1일 자동으로 정산됩니다.
