## 1단계: 계정 만들기

[castfolio.app/login](https://castfolio.app/login)에서 제작 파트너 계정을 만듭니다.

<Note>
  Castfolio는 방송인 PR 페이지를 제작·납품하는 **에이전트(제작 파트너)**를 위한 서비스입니다. 방송인 본인이 직접 가입하는 서비스가 아닙니다.
</Note>

## 2단계: 방송인 등록

대시보드에서 **방송인 추가** 버튼을 눌러 기본 정보를 입력합니다.

| 항목 | 예시 |
|------|------|
| 이름 | 홍길동 |
| 포지션 | 쇼호스트 |
| 연락처 | 010-0000-0000 |

## 3단계: 자료 수집

두 가지 방법 중 선택합니다.

<Tabs>
  <Tab title="방송인 직접 제출">
    제출 링크를 생성하여 방송인에게 카카오톡 또는 이메일로 발송합니다.
    방송인이 직접 이력, 방송 경력, 사진 등을 업로드합니다.
  </Tab>
  <Tab title="담당자 대행 입력">
    담당자가 방송인에게 자료를 받아 직접 입력합니다.
    자료가 많을 때 유용합니다.
  </Tab>
</Tabs>

## 4단계: 테마 선택 & 편집

7가지 전문 테마 중 하나를 선택합니다.

<CardGroup cols={3}>
  <Card title="앵커 클린" icon="circle" color="#FFFFFF">
    흰 배경의 깔끔한 뉴스 앵커 스타일
  </Card>
  <Card title="프레스티지 블랙" icon="circle" color="#1A1A1A">
    고급스러운 블랙 계열 프리미엄 스타일
  </Card>
  <Card title="럭스 마블" icon="circle" color="#C9B99A">
    대리석 텍스처의 럭셔리 스타일
  </Card>
  <Card title="프레시 스카이" icon="circle" color="#7DD3FC">
    밝고 경쾌한 하늘색 스타일
  </Card>
  <Card title="소프트 블러시" icon="circle" color="#FBBF9A">
    따뜻한 핑크 계열 감성 스타일
  </Card>
  <Card title="내추럴 그린" icon="circle" color="#86EFAC">
    자연스러운 그린 계열 스타일
  </Card>

빌더에서 섹션을 추가·삭제·순서 변경하여 커스터마이징합니다.

## 5단계: 검토 링크 발송

<Steps>
  <Step title="미리보기 생성">
    빌더 상단의 **검토 링크 생성** 버튼을 클릭합니다.
  </Step>
  <Step title="링크 발송">
    생성된 토큰 기반 비공개 링크를 방송인에게 전달합니다.
  </Step>
  <Step title="피드백 반영">
    방송인의 수정 요청을 빌더에서 반영합니다.
  </Step>
</Steps>

## 6단계: 배포 & 납품

검토 완료 후 **배포** 버튼을 누르면 자동으로 생성됩니다.

- 공개 PR 페이지 URL
- 고해상도 QR 코드 (PNG/SVG)
- PDF 명함 (A4/명함 사이즈)

<Check>
  첫 번째 PR 페이지 납품을 완료했습니다. 매월 1일에 수익의 85%가 자동 정산됩니다.
</Check>
